Project/Product Manager

In addition to my duties as a project manager, I assumed the role as a product manager where I contributed to the overall design of the application. Iteration 1 of LiveIn acts as a tool to delegate chores amongst roommates using an admin based distribution platform, where he/she can create chores with a name and difficulty rating. Users can take chores on their own or be assigned them by an admin. Once a user marks a chore as complete, the admin can review the completeness of the chore. If the admin is satisfied, the chore will be approved and the user will be allocated the difficulty points. If denied, the admin has the ability to write a small message indicting what the problem is, then the user will recieve an email shortly after admin submission with the message. Please review the Portfolio documents to gain a better understanding of how LiveIn operates. We plan to release this application on the Google Play Store in the summer of 2015.
